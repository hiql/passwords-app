# Passwords

A random password generator

![alt text](screenshot.png)

## Features

- Generator
  - Random
  - Memoriable
  - PIN
- Hasher
  - MD5
  - Bcrypt
  - SHA1/SHA224/SHA256/SHA384/SHA512
  - BASE64
- Analyzer
  - Password strength test
  - Common password check
  - Estimate the time to crack
